# Privacy Policy

## Data Collection

This MCP server:
- Runs entirely locally on your Mac
- Communicates only with Apple's App Store Connect API (api.appstoreconnect.apple.com)
- Does not collect or transmit any personal data to third parties
- Requires an App Store Connect API key for authentication

## Data Storage

No data is stored by this extension beyond JWT tokens cached in memory during the session.

## Third-Party Services

This extension communicates with Apple's App Store Connect API using your provided credentials.
